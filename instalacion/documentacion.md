# Documentación



## En local

- **[Generada con Gitbook](https://www.gitbook.com)**
- Para generar las slides, pdf, epub o mobi
- Es necesario tener instalado calibre

```
git clone https://github.com/juanda99/node-mongodb
cd node-mongodb
node_modules/.bin/gitbook install
npm install
npm run slides
npm run pdf
npm run epub
npm run mobi
```
